Config File Location

MangoHud reads its config from:

~/.config/MangoHud/MangoHud.conf

If the file doesn't exist, create it. MangoHud picks it up on the next game launch.

For per-game configs, name the file after the game's executable:

~/.config/MangoHud/<application_name>.conf
Tip
Find the executable name by checking the game's process while it's running: ps aux | grep -i <game>

Config Syntax

One parameter per line. Boolean parameters are enabled just by listing them with no value. Parameters that take a value use =.

fps
gpu_stats
font_size=20
position=top-left

Lines starting with # are comments.

Starter Config

A minimal config that provides useful data without clutter:

toggle_hud=Shift_R+F12
fps
gpu_stats
gpu_temp
cpu_stats
cpu_temp
ram
position=top-left
font_size=20
background_alpha=0.4

Keybinds

ParameterDefaultDescription
toggle_hudShift_R+F12Show/hide overlay
toggle_loggingShift_L+F2Start/stop logging
upload_logShift_L+F12Upload log to FlightlessMango
reload_cfgShift_L+F4Reload config without restarting

Custom keybinds use the format Modifier+Key. Modifiers: Shift_L, Shift_R, Control_L, Control_R.

FPS Limiting

fps_limit=60

Multiple limits can be cycled through:

fps_limit=30,60,120,0

Where 0 means unlimited. Each press of the toggle cycles to the next value.

Logging

MangoHud can log performance data to CSV for later analysis.

output_folder=/home/youruser/mangologs
log_duration=60
autostart_log
ParameterDescription
output_folderDirectory where logs are saved
log_durationMax log duration in seconds (0 = unlimited)
autostart_logBegin logging immediately on launch
log_intervalMilliseconds between log entries (default: 100)

Logs can be uploaded to flightlessmango.com for visualization, or opened in any spreadsheet application.

Applying Changes

Three options:

1. Press Shift_L+F4 in game to reload the config file live.

2. Restart the game.

3. For Steam launch option changes, restart the game.

Troubleshooting

If MangoHud isn't appearing, confirm it's being invoked. Your Steam launch options should contain either mangohud %command% or MANGOHUD=1 %command%.

If the config file isn't being read, verify the path is exactly ~/.config/MangoHud/MangoHud.conf (case sensitive).

Common Issue
If you're using Flatpak Steam, MangoHud needs to be installed inside the Flatpak sandbox as well, or the overlay won't load.

To verify MangoHud is loaded and reading your config, launch the game from a terminal and look for MangoHud output lines confirming the config path.
