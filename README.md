# MangoHud Configuration Guide

A single-page reference for setting up and customizing [MangoHud](https://github.com/flightlessmango/MangoHud), the Vulkan/OpenGL performance overlay for Linux.

## What's Covered

- Config file location and per-game configs
- Config syntax
- Starter config ready to copy
- Full parameter reference for display, metrics, keybinds, FPS limiting, and logging
- Three example configs (minimal, full monitoring, headless logging)
- Custom theme ([Junkyard Theme](https://github.com/ihpsm/junkyard/blob/main/MangoHud.conf))
- Troubleshooting common issues

## Live Site

Hosted via GitHub Pages:

```
[https://<username>.github.io/<repo-name>/](https://undeadbulwark.github.io/MangoHud-Guide/
```

## Deploy

1. Fork or clone this repo.
2. Go to **Settings > Pages**.
3. Set source to `main` branch, root directory.
4. The site goes live at the URL above within a minute or two.

No build step. No dependencies. Just a single `index.html`.

## Local Preview

Open `index.html` in a browser. That's it.
